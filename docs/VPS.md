# VPS Debian/Ubuntu

Execute o instalador como root ou como usuário com `sudo`. O FizLab usa a porta
`8080` por padrão mesmo em VPS, evitando conflito com outros servidores web.

Antes de publicar o dashboard na internet, configure autenticação, HTTPS e
firewall. A versão atual foi projetada para rede privada e é somente leitura.
